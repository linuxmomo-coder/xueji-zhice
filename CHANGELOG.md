# Changelog

## 0.2.2 - 生产缺陷整改

### 数据正确性
- 练习选题强制同时匹配学生真实年级与科目，禁止跨年级出题。
- 管理员题库页面使用接口 `meta.total` 显示真实总数，不再把第一页数量当总数。
- 移除“生产数据库”等无法由前端确认的误导性文案，更新时间明确为接口生成时间。

### 用户闭环
- 家长可以为未绑定账号的学生创建学生登录账号，并自动建立家庭成员关系。
- 前端新增学生账号绑定表单、真实错题记录列表、资料记录列表和家长资料确认界面。
- 资料确认结果写入数据库并保留审计记录；未启用OCR时明确显示人工确认状态。

### 安全与会话
- 前端会话从长期 localStorage 迁移到 sessionStorage。
- Access Token 过期后自动轮换 Refresh Token，刷新失败会清除本地会话。
- 生产 Nginx 关闭 `/docs`、`/redoc` 和 `/openapi.json`，增加 CSP 等安全响应头。

### 数据库与构建
- 初始数据库基线迁移改为不可逆，禁止 downgrade 直接删除全部业务表。
- CI 使用 PostgreSQL/pgvector 容器验证 Alembic 升级，不再以 SQLite 迁移结果替代生产验证。
- 前端构建明确使用公共 npm registry，前后端与 Docker 镜像构建继续作为合并门禁。

### 测试
- 新增跨年级出题回归测试。
- 新增家长创建并绑定学生账号、学生实际登录和本人数据隔离测试。

## 0.2.1 - 角色登录与实时数据界面

### 登录与权限
- 登录页新增家长、学生、管理员身份选择。
- 后端同时校验账号、密码和所选角色，角色不匹配时拒绝登录。
- 登录后导航和功能由服务器返回的真实账号角色决定，不提供角色切换。
- 移除登录页预填演示账号和演示密码。

### 生产数据界面
- 首页统计全部改为数据库实时聚合，不再展示固化数量。
- 已完成练习只统计 completed 会话，错题只统计未掌握记录。
- 提示信息根据学生、练习、错题、资料和题库实际状态生成。
- 练习科目和题量从学生年级对应的已审核发布题库实时读取。
- 学生档案、题库和空状态均以服务器实际返回为准。

### 测试
- 新增角色选择与账号角色不匹配测试。
- 新增工作台数据库实时计数测试。

## 0.2.0 - 安全基线与真实学习闭环

### 安全与身份
- 新增家长注册、登录、访问令牌、刷新令牌轮换和退出。
- 新增家庭、家庭成员、学生与登录账号关系。
- 所有学生业务接口校验当前用户与家庭/学生关系。
- 管理员、家长、学生权限由后端令牌决定，前端不再切换角色。
- 统一错误协议与 request_id。

### 生产基线
- 生产环境默认密钥、SQLite、通配 CORS、demo 和自动建表将触发启动失败。
- demo 路由和演示种子只在开发环境显式启用。
- 引入 Alembic 迁移；应用启动不再依赖 create_all。
- Docker 后端启动前自动执行数据库迁移。

### 题库与练习
- 题目身份与题目版本分离。
- 新增选项、作答字段、结构化判分规则、题目图片资产表。
- 练习保存题目版本和题目快照。
- 新增答题记录、归一化答案、判分明细和错题记录。
- 支持全角半角、大小写配置、数值容差和根式符号等价判分。

### 资料与审计
- 文件上传校验 MIME、大小和 SHA-256，保存到家庭隔离私有目录。
- 重复文件识别；家长确认资料时写入审计事件。
- 增加本地私有存储适配层，为 MinIO/COS 预留统一接口。

### 前端
- 新增真实登录/家长注册界面。
- 根据登录身份显示家长、学生或管理员工作台。
- 新增学生建档、资料上传、短练习和题库查看操作界面。
- 移除原始 API JSON 展示。

### 质量
- 新增生产配置、认证、跨家庭隔离、练习错题和数学等价判分测试。
- CI 增加 Ruff、Bandit、Alembic 升降级、覆盖率、前端构建和 Docker 构建。
