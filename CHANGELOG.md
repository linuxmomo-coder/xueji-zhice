# Changelog

## 0.2.0 - 安全基线与真实学习闭环

### 安全与身份
- 新增家长注册、登录、访问令牌、刷新令牌轮换和退出。
- 新增家庭、家庭成员、学生与登录账号关系。
- 所有学生业务接口校验当前用户与家庭/学生关系。
- 管理员、家长、学生权限由后端令牌决定，前端不再切换角色。
- 统一错误协议与 request_id。

### 生产基线
- 生产环境默认密钥、SQLite、通配 CORS、demo 和自动建表将触发启动失败。
- demo 路由和演示种子只在开发环境显式启用。
- 引入 Alembic 迁移；应用启动不再依赖 create_all。
- Docker 后端启动前自动执行数据库迁移。

### 题库与练习
- 题目身份与题目版本分离。
- 新增选项、作答字段、结构化判分规则、题目图片资产表。
- 练习保存题目版本和题目快照。
- 新增答题记录、归一化答案、判分明细和错题记录。
- 支持全角半角、大小写配置、数值容差和根式符号等价判分。

### 资料与审计
- 文件上传校验 MIME、大小和 SHA-256，保存到家庭隔离私有目录。
- 重复文件识别；家长确认资料时写入审计事件。
- 增加本地私有存储适配层，为 MinIO/COS 预留统一接口。

### 前端
- 新增真实登录/家长注册界面。
- 根据登录身份显示家长、学生或管理员工作台。
- 新增学生建档、资料上传、短练习和题库查看操作界面。
- 移除原始 API JSON 展示。

### 质量
- 新增生产配置、认证、跨家庭隔离、练习错题和数学等价判分测试。
- CI 增加 Ruff、Bandit、Alembic 升降级、覆盖率、前端构建和 Docker 构建。
