## 状态标记

- **available**：当前代码和迁移已实现，可进行测试；
- **partial**：核心结构可用，但仍需补充完整后台或适配器；
- **planned**：已设计，未在本版本交付；
- **excluded**：当前产品边界明确不做。

API文档中的状态以实际路由为准；OpenAPI是当前接口的机器可读事实来源。界面统计和可用操作必须以当前数据库和登录权限为准，不得使用固定演示值填充生产界面。
