# 后续升级：题目勘误与相似题型推荐

> 状态：planned；当前版本仅保留数据和服务扩展点。

## 1. 题目勘误

学生可在做题、解析和错题页提交问题类型、说明、建议答案及是否影响判分。

状态：

```text
student_submitted → ai_reviewing
→ likely_invalid | likely_valid | uncertain
→ admin_reviewing → rejected | corrected | retired
```

AI必须独立求解并检查条件、答案、解析、单位和标签。高置信疑似错误题先`suspended`；正式修订只发布新版本。影响判分时需要创建重判任务，修正attempt、wrong_question、掌握度、奖励和报告，并完整审计。

## 2. 题型分类

不能只用单选/填空。至少包含：题型族、题型模板、核心知识点、考查技能、错误模式、认知层级、难度特征和表达形式。

AI标签只能作为建议，保存模型、置信度和来源，经审核后使用。

## 3. 学生错误画像

满足最低证据量后才形成弱项，例如最近3题错2题、连续错2次或复测失败。记录正确率、提示、用时、错误模式、同类/变式通过情况和下次推荐时间。

## 4. 推荐顺序

```text
原题订正 → 同模板低难度 → 同模板正常难度
→ 不同情境同类题 → 变式题 → 综合应用
```

结构化标签做硬过滤，向量只辅助排序。连续通过2—3道非重复题后降低频率；持续失败则降难度并回退前置知识点。
