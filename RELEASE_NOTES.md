# 学迹智评 v0.3.0 发布说明

v0.3.0 是首个生产候选版本。它不再依赖角色切换、固定统计或模拟业务数据；登录身份、题库、练习、资料、OCR、AI报告、勘误与历史重判均由真实后端、数据库和权限控制。

## 可上线功能

- 家长注册、邮箱验证、密码找回、会话轮换和账号停用；
- 家长、学生、管理员按真实角色登录；
- 家庭数据隔离和监护人授权；
- 家长创建学生档案并绑定学生账号；
- Excel题库导入、逐行校验、来源版权、审核、发布和题图资产；
- 按学生实际年级和科目生成练习；
- 选择题、文本、数值容差、分数和基础数学表达式判分；
- 练习快照、答题记录、错题、原题复测；
- 学习资料私有上传、OCR异步任务、家长修改和确认；
- 百炼主模型、混元备用模型的证据化AI报告；
- 题目勘误、修正版、历史重判、通知、题型标签和同类题推荐；
- S3/COS兼容私有对象存储；
- Alembic 0001—0009显式迁移；
- OCR Worker、AI Worker和历史重判Worker；
- 生产配置强校验、部署前备份、预检、失败回退和上线后冒烟测试。

## 上线边界

以下能力不属于v0.3.0首发范围，界面和文档不得将其描述为已提供：

- 整张试卷自动拆题；
- 手写作业答案识别；
- 纸质错题自动提取；
- 高风险主观题自动评分；
- 教师端、班级管理、公开排名和社交功能；
- 完整教材知识图谱和向量相似检索；
- 多级管理员RBAC。

## 生产环境必须配置

- PostgreSQL、Redis；
- 私有S3兼容存储或腾讯云COS；
- SMTP邮件服务；
- PaddleOCR HTTPS服务；
- 百炼API Key，可选混元备用Key；
- 长度不少于32位的随机SECRET_KEY；
- 明确CORS域名和HTTPS公网地址；
- 可信公网TLS终止层，例如腾讯云负载均衡、Caddy或Traefik。

部署前必须执行：

```bash
sh deployment/preflight.sh
```

上线后必须执行：

```bash
BASE_URL=https://your-domain.example sh deployment/smoke-test.sh
```

未通过预检、CI或冒烟测试时，不得切换正式流量。
